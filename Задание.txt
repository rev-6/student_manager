Django-based Student Management System
Задание
Цель: Создать веб-интерфейс для управления студентами с использованием фреймворка Django, включая админ-панель и учет рабочего времени.

        Основные Требования
Админ-панель:
    Управление студентами: создание, редактирование, удаление студентов, привязка анкет в Redmine и GitLab, установка статусов, добавление фото.
    Управление администраторами: создание, редактирование, удаление учетных записей.
    Фильтрация и поиск студентов.
    Чтение сообщений от студентов.
    Управление правилами пользования ПК.
    Просмотр студентов на рабочем месте.

Панель студентов:
    Регистрация и аутентификация через Django Authentication System.
    Тайм-трекер с авто-выключением (можно реализовать через Celery).
    Просмотр связанных данных.
    Чтение правил пользования ПК.
    Отправка сообщений администратору.

Публичный доступ:
    Страница с текущим списком работающих студентов, без необходимости входа в систему.
    Хранение данных:
    Все данные студентов — в базе данных SQLite (или PostgreSQL).
    Учетные записи администраторов и пароли – через Django ORM в базе данных.
    Фото студентов – хранение в файловой системе, ссылки на них в базе данных.

Нефункциональные требования:
    Производительность: Быстрый доступ и минимальное время отклика.
    Безопасность: Защита паролей, данные хранятся в зашифрованном виде.
    Интерфейс: Удобный, интуитивный, адаптивный через Bootstrap или Tailwind CSS.
    Расширяемость: Микросервисная архитектура при необходимости.
    Архитектура системы.

Frontend:
    Языки: HTML, CSS, JavaScript.
    Фреймворк: Django Templates (или дополнительно использовать React/Vue для динамичных компонентов).
    Компоненты: Админ-панель, Пользовательская панель, Публичная страница, \
    Модальные окна, Страница входа/регистрации.

Backend:
    Сервер: Django.
    Данные студентов: Хранение в базе данных через Django ORM.
    Учетные записи администраторов: Django Authentication System.
    API: Django REST Framework для построения интерфейса API.
